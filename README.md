# Sales Dashboard (Next.js 15)

## 🚀 Overview
A simple sales dashboard built using Next.js 15, TypeScript, Tailwind CSS, and Recharts.

## 📊 Features
- Sales data for 2022, 2023, 2024
- Multiple chart types (Bar, Line, Pie)
- Dynamic chart switching
- Custom sales threshold filter
- Responsive UI

## 🛠 Tech Stack
- Next.js 15
- TypeScript
- Tailwind CSS
- Recharts

## 📦 Setup

```bash
npm install
npm run dev
